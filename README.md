# TechHub Academy

A responsive educational platform developed using HTML5, CSS3, JavaScript and Bootstrap 5.

## Project Goals

The website provides information about technology courses, instructors, student success stories, learning resources, and contact information.

## Planned Features

- Responsive design
- Course catalog
- Instructor profiles
- Blog section
- Gallery
- FAQ page
- Contact form validation
- Dark mode toggle

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Git & GitHub
